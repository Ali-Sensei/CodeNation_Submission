Flappy Bird Clone

Based off the viral game 'Flappy Bird' by Dong Nguyen of dotGears, this game was created using the Construct game engine following a Udemy tutorial. While the sprites and sounds were shared by the Udemy tutor, I followed the tutorials to create the logic of the game.

This game was intended for mobile devices. To play, simply use the mouse to tap the screen!
